### Usage

Fork and clone the Chatty App

```
git clone git@github.com:ChristinaHsu88/Chatty-App.git
cd chattyApp
```
Install the dependencies on package.json file and start the server (see the following):

```
npm install
npm start
open http://localhost:3000
```

### Dependencies 
* React
* React-dom
* Webpack
